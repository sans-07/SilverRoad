# SilverRoad

어르신 안심 귀가 및 보호자 모니터링 서비스입니다.

## 기술 스택

- **Frontend**: React, Vite, Leaflet, Firebase
- **Backend**: Node.js, Express, Firebase Admin

## 설치 및 실행

### Server
루트 디렉토리에서 실행:
```bash
npm install
npm start
```

### Client
`client` 디렉토리에서 실행:
```bash
cd client
npm install
npm run dev
```

## 환경 변수

루트와 `client` 디렉토리에 각각 `.env` 파일이 필요합니다.
